# 🤖 Perzaan Gaming Aternos 24/7 Hosting Bot

A Minecraft bot that helps keep an Aternos server online 24/7 by automatically joining it using a Mineflayer-based bot. Perfect for SMPs or small multiplayer servers that shut down when no players are online.

---

## ✨ Features

* 🚀 **Auto Connect System**
  Automatically joins your Aternos server without any manual work.
* ♾️ **24/7 Uptime (Anti-AFK)**
  Smart anti-AFK system (jump/move) to keep your server always online.
* 🔄 **Advanced Auto-Reconnect**
  Instantly reconnects if the bot gets kicked, server restarts, or internet drops.
* ☁️ **Free 24/7 Hosting Support**
  Fully optimized for platforms like Render to keep your bot running non-stop.
* 🔌 **Plugin Friendly (Auto Auth)**
  Works perfectly with Paper, Spigot, Bukkit servers with built-in auto-login/register.
* ⚡ **Lightweight & Fast**
  Low CPU/RAM usage — runs smoothly even on low-end systems.
* 🛠️ **Easy Setup (Beginner Friendly)**
  Simple config file, no coding knowledge required.
* 💬 **Auto Chat / Activity (Optional)**
  Can send messages or perform actions to avoid inactivity detection.

## ⚙️ Usage
*   **Start**: Just turn on your Aternos server. The bot will join automatically.
*   **Status**: Visit the Render URL to see a status dashboard.
*   **Chat**: The bot logs chat to the console.
